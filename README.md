# Backend File manager

This is a [Node.js](https://nodejs.org/) backend for file manager project.

## Getting Started

Clone de project:

```bash
git clone https://github.com/raulglezrdguez/backend-file-manager.git
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```
